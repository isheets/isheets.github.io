PROJECTS

project 1

Project 1 is an interactive narrative. I chose to create mine on the subject of driving. I personally do not like being in a car ever, and I have found that using alternative forms of transportation has made my life easier and generally better. The content of my narrative is a mix of factual information and anecdotal experience. In terms of user interactivity, my argument is a single page with elements and backgrounds changing as the user scrolls. I used a library called ScrollMagic with a plugin called GreenSock Animation Platform to handle the animations and scrolling.

project 2

For project 2 I used APIs to add on to my content from the first project. I wanted to create a tool that enabled the user to visualize of how long it would take them to get places they usually drive to, so I used an API called Targomo to request some iso-time polygons for walking, biking, or using public transportation based on a user inputted address. I used Google Places API for address autocomplete as well as geocoding. I also made slight modifications to the content from my first project to create a slightly more balanced arguement. Additionally, I included a navigation at the top so the user can go straight to the mapping tool or read the content first.
